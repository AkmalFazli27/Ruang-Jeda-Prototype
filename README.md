# Ruang Jeda 🧘

Aplikasi kesehatan mental interaktif untuk membantu pengguna mengelola stres, kecemasan, dan kesehatan mental secara mandiri. Dilengkapi dengan jurnal emosi, teknik pernapasan, musik relaksasi, dan akses konseling.

## Fitur

- **📝 Jurnal Emosi** — Menulis perasaan dan mendapatkan analisis kondisi mental (Tier 1-3)
- **🔍 Triage Cerdas** — Deteksi otomatis tingkat urgensi berdasarkan tulisan pengguna
- **🌬️ Latihan Pernapasan 4-7-8** — Teknik pernapasan untuk menenangkan diri dengan panduan visual
- **🎵 Musik Relaksasi** — Koleksi musik dan suara alam untuk terapi relaksasi
- **🤝 Konseling** — Akses ke layanan konseling BKM (Biro Konseling Mahasiswa)
- **📊 Riwayat & Profil** — Pantau perkembangan kondisi mental dari waktu ke waktu

## Tech Stack

- **Framework**: React 18 + TypeScript
- **Build Tool**: Vite 6
- **Styling**: Tailwind CSS 4
- **Animation**: Motion (Framer Motion)
- **UI Library**: Radix UI, Lucide Icons
- **Deployment**: Vercel

## Cara Menjalankan

```bash
# Install dependencies
npm install

# Jalankan development server
npm run dev

# Build untuk production
npm run build
```
